Md's Basic Grammar

Veröffentlicht am 2020-01-24 Bearbeitet am 2020-01-25

Title

Add # before the text you want set it as title.
The number of symbols indicate the level of the title.One for first, and two for second. This is rule of nature.
note: The standard grammar is usually followed by a space.

Typeface

  • Bold
    add two * around the text
  • Italics
    add one * around the text
  • Italics Bold
    add three * around the text
  • Delete Line
    add two ~ around the text

Quote

Add symbol > before the text you wanner quote.What’s more, quoting can be nested.

Faker once said: When you lose, whatever you said all seems like making excuses.

It can be nested!

Separator

Three or more - or * gathering in a blank row!
